Learning React
=========================

It's a little project to explore react's world

## Installation

```
npm install
```

## Run
```
npm start
```

## Changelog 

### v0.1
Features:

    - Redux
        - Action creators
        - Reducers
        - Store
    - Patterns:
        - Container presentational components
        - HOC
    - React Bootstrap
    - Recompose
    - ESlint
